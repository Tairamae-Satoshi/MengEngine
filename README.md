# README

This is a skeleton animation engine. And a procedural character locomotion system with simple control is implemented in this engine...



#### Features:

* High-level Animation

  - [x] Animation sampling
  - Animation blending
    - [x] blending
    - [ ] Partial animation blending
    - [ ] Additive blending

  - [x] Linear skinning
  - [x] Motion analysis

  - IK
    - [x] Two bone IK
    - [x] Aim IK

* Character Control

  - [ ] Root Motion
  - [x] Spring Damper
